Patch size is a value that states how big the image is in the prediction process. When making a prediction, the image will be cropped to a smaller size for greater accuracy. For example, the value of the patch size is 800, then the cut of each image is 800 x 800 pixels. Note that the patch size value cannot exceed the predicted total pixel size of the image.

You can try changing the value of the patch size. This application uses Mask RCNN model for detecting the buildings in satellite images. The model will extract the detected building and mark the position using bounding box. Then the application will estimate the geolocation (longitude and latitued) of each building in the image and display the results.

This application will accept TIFF (*.tif or *.tiff) image or raster as an input. TIFF raster usually contains coordinate system information in the image. This information will help the application to estimate the real world position of the detected buildings.
